# AI4ALL-19D
AI4ALL Group 19D Project Repo

## Setup Instructions
1. After creating your virtual environment, in your terminal, run: <code>pip install -r requirements.txt</code>
2. Download the dataset from https://www.kaggle.com/competitions/aptos2019-blindness-detection
3. Place it in the `data/raw/` folder
4. To train first model, run <code>python model/train_eval.py</code>
